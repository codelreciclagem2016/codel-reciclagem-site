# Codel Reciclagem — Site Institucional

Site institucional da **Codel Reciclagem**, desenvolvido com HTML, CSS e JavaScript puro (sem frameworks), pronto para hospedagem estática na Vercel.

## Estrutura

```
codel-reciclagem-site/
├── index.html          # Página principal
├── data/
│   └── content.json    # Conteúdo editável do site
├── vercel.json         # Configuração da Vercel
└── README.md
```

## Seções do site

- Hero com contadores animados
- Números de impacto (10+ anos, 105,5t recicladas, 8 pontos)
- Serviços oferecidos
- Materiais aceitos
- Vídeo institucional (YouTube)
- Pontos de coleta
- FAQ
- Formulário de contato (envia via WhatsApp)
